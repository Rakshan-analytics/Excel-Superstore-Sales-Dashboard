# 📊 Superstore Sales Performance Dashboard (Excel)

## 📌 Project Overview

This project is an interactive **Sales Performance Dashboard** built using **Microsoft Excel**. It analyzes Superstore sales data and transforms raw data into meaningful business insights through interactive visualizations and KPI reporting.

The dashboard allows users to filter data dynamically using slicers and monitor sales performance across different business dimensions.

### 📊 Visualizations

- Monthly Profit Trend

- Profit by Category

- Sales & Profit by Sub-Category
- Sales & Profit by State
- Top Customers (Sales & Profit)

---

## 💡 Key Business Insights

- Technology generated the highest overall sales and profit.
- November, September, and December consistently recorded the highest sales across multiple years.
- January and February experienced comparatively lower sales.
- The West region generated the highest sales among all regions.
- California recorded the highest sales among all states.
- Copiers and Phones were among the best-performing sub-categories.
- Some states generated negative profit despite strong sales, indicating potential opportunities for cost optimization.

## 🚀 How to Use

1. Download the Excel workbook.
2. Open it in Microsoft Excel.
3. Navigate to the **Dashboard** sheet.
4. Use the slicers (Region, Category, Segment, Ship Mode) to interact with the dashboard.
5. Explore KPIs and charts to analyze sales performance.
